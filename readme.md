# ðŸ“ˆ Stock Market Prediction Web App (Real-Time + ML)

A **Stock Market Analysis and Prediction Web App** built using **FastAPI** and **React**. The platform fetches **real-time stock data**, performs **7-day price forecasting** using **ARIMA, LSTM, and Linear Regression**, and uses **real news sentiment analysis** to recommend BUY/SELL/HOLD actions.
